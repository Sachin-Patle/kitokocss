KitokoCSS v1.0

Framework CSS moderne, esthétique et performant

Développé par FomaDev • Publié le 27 novembre 2025

Installation via CDN

Copiez et collez ces liens dans le <head> de votre page HTML :

<!-- CSS --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/FomaDev/KitokoCSS@v1.0/dist/kitoko.min.css">
<!-- JavaScript (optionnel, pour composants interactifs) --> <script src="https://cdn.jsdelivr.net/gh/FomaDev/KitokoCSS@v1.0/dist/kitoko.bundle.min.js"></script>

Démarrage Rapide

Template HTML de base pour commencer :

<!DOCTYPE html> <html lang="fr"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Mon Projet KitokoCSS</title> <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/FomaDev/KitokoCSS@v1.0/dist/kitoko.min.css"> </head> <body> <div class="kt-container"> <h1>Bienvenue dans KitokoCSS</h1> <button class="kt-btn kt-btn-primary">Commencer</button> </div> <script src="https://cdn.jsdelivr.net/gh/FomaDev/KitokoCSS@v1.0/dist/kitoko.bundle.min.js"></script> </body> </html>

Fonctionnalités Principales

  • Système de Grille Responsive : 12 colonnes, 6 breakpoints (xs à xxl)
  • Composants UI Complets : Boutons, Cartes, Formulaires, Modales, Navigation, etc.
  • Classes Utilitaires : Espacement, Flexbox, Couleurs, Bordures, Ombres
  • JavaScript Léger : Vanilla JS, aucune dépendance
  • Design Moderne : Interface "Kitoko" (belle) par défaut
  • Préfixe kt- : Toutes les classes utilisent le préfixe kt-

Exemples de Code

Bouton

<button class="kt-btn kt-btn-primary">Cliquez-moi</button>

Grille

<div class="kt-container"> <div class="kt-row"> <div class="kt-col-md-6">Colonne 1</div> <div class="kt-col-md-6">Colonne 2</div> </div> </div>

Carte
