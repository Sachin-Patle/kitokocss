Framework CSS moderne, esthétique et performant

Développé par FomaDev • Publié le 27 novembre 2025

Introduction

KitokoCSS est un framework CSS moderne, esthétique et hautement performant, conçu pour accélérer le développement front-end sans compromettre le design. Fini les interfaces banales, créez des applications véritablement "Kitoko" (belles).

Philosophie

  • Design-First & Esthétique : Tous les composants ont un design par défaut moderne, épuré et attrayant
  • Facilité d'Utilisation : Syntaxe simple et intuitive avec le préfixe kt-
  • Performance : Code léger, optimisé et modulaire
  • Responsivité Universelle : Gestion de tous les breakpoints, du mobile aux écrans 8K

Pourquoi KitokoCSS ?

KitokoCSS offre une alternative moderne aux frameworks existants avec :

  • Un système de nommage cohérent avec le préfixe kt-
  • Une architecture modulaire SCSS pour une personnalisation facile
  • Des composants entièrement stylisés et responsifs par défaut
  • Un JavaScript compagnon léger (Vanilla JS, aucune dépendance)
  • Une documentation complète et des exemples prêts à l'emploi

Installation

Via CDN (Recommandé)

La méthode la plus simple pour démarrer rapidement :

<!-- CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/fomadev/kitokocss@v1.0.0/dist/kitoko.min.css">

<!-- JavaScript (optionnel, pour les composants interactifs) -->
<script src="https://cdn.jsdelivr.net/gh/fomadev/kitokocss@v1.0.0/dist/kitoko.bundle.min.js"></script>

Installation Locale

Pour une utilisation en développement ou pour personnaliser le framework :

# Cloner le repository
git clone https://github.com/FomaDev/KitokoCSS.git
cd KitokoCSS

# Installer les dépendances
npm install

# Compiler le CSS
npm run build

Système de Grille

KitokoCSS utilise un système de grille flexible basé sur Flexbox, avec 12 colonnes et 6 breakpoints responsives.

Breakpoints

Breakpoint Largeur Description
xs < 576px Extra Small (téléphones)
sm ≥ 576px Small (téléphones en paysage)
md ≥ 768px Medium (tablettes)
lg ≥ 992px Large (desktop)
xl ≥ 1200px Extra Large (grand desktop)
xxl ≥ 1400px Double Extra Large (écrans 4K+)

Conteneurs

Utilisez .kt-container pour une largeur fixe responsive, ou .kt-container-fluid pour une largeur 100% :

<div class="kt-container">
  <!-- Contenu avec largeur max selon breakpoint -->
</div>

<div class="kt-container-fluid">
  <!-- Contenu pleine largeur -->
</div>

Lignes et Colonnes

Créez des lignes avec .kt-row et des colonnes avec .kt-col-* :

<div class="kt-container">
  <div class="kt-row">
    <div class="kt-col-md-6">Colonne 1</div>
    <div class="kt-col-md-6">Colonne 2</div>
  </div>
</div>
Colonne 1 (50%)
Colonne 2 (50%)

Colonnes Responsives

Vous pouvez spécifier différentes largeurs selon les breakpoints :

<div class="kt-col-12 kt-col-md-6 kt-col-lg-4">
  <!-- 100% sur mobile, 50% sur tablette, 33% sur desktop -->
</div>

Variables Sass

Toutes ces couleurs sont disponibles sous forme de variables Sass dans le fichier src/config/_variables.scss :

// Couleurs du thème
$kt-primary: #007bff;
$kt-secondary: #6c757d;
$kt-success: #28a745;
$kt-danger: #dc3545;
$kt-warning: #ffc107;
$kt-info: #17a2b8;
$kt-light: #f8f9fa;
$kt-dark: #343a40;

// Carte des couleurs du thème
$kt-theme-colors: (
  "primary": $kt-primary,
  "secondary": $kt-secondary,
  "success": $kt-success,
  "danger": $kt-danger,
  "warning": $kt-warning,
  "info": $kt-info,
  "light": $kt-light,
  "dark": $kt-dark
);

JavaScript

KitokoCSS inclut un JavaScript compagnon léger (Vanilla JS) pour les composants interactifs.

Initialisation Automatique

Le JavaScript s'initialise automatiquement au chargement de la page. Assurez-vous d'inclure le script :

<script src="https://cdn.jsdelivr.net/gh/fomadev/kitokocss@v1.0.0/dist/kitoko.bundle.min.js"></script>

API JavaScript

Vous pouvez également utiliser l'API JavaScript directement :

// Modal
const modal = new kitoko.Modal('#myModal');
modal.show();
modal.hide();

// Navbar Toggler
const toggler = new kitoko.Toggler('[data-kt-toggle="collapse"]');
toggler.toggle();

// Carousel
const carousel = new kitoko.Carousel('#myCarousel', {
  interval: 3000,
  wrap: true
});

// Toast
const toast = new kitoko.Toast('#myToast', {
  autohide: true,
  delay: 5000
});
